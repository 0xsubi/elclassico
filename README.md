# El Classico — Menu

## How it works

```
Google Sheets (you edit)
       ↓
GitHub Actions pulls the CSV  ← runs daily at 6 AM IST, or trigger manually
       ↓
Regenerates index.html and commits both files
       ↓
GitHub Pages serves the updated site (~30s later)
```

You never touch the repo after initial setup. Just edit the sheet.

---

## Initial setup

### 1. Publish your Google Sheet as CSV

- Open your sheet → **File → Share → Publish to web**
- Select your sheet tab, change format to **Comma-separated values (.csv)**
- Click **Publish** → copy the URL

It looks like:
```
https://docs.google.com/spreadsheets/d/SHEET_ID/pub?gid=0&single=true&output=csv
```

### 2. Add the URL as a GitHub secret

- In your repo → **Settings → Secrets and variables → Actions → New repository secret**
- Name: `SHEETS_CSV_URL`
- Value: the URL you copied above

### 3. Push this repo to GitHub

```bash
git init
git add .
git commit -m "Initial deploy"
git branch -M main
git remote add origin https://github.com/<you>/<repo>.git
git push -u origin main
```

### 4. Enable GitHub Pages

- **Settings → Pages → Source: Deploy from branch → main / (root)**
- Live at: `https://<you>.github.io/<repo>/`

### 5. Run the workflow once manually

- Go to **Actions → Sync menu from Google Sheets → Run workflow**
- This does the first sync immediately without waiting for the schedule

---

## Day-to-day usage

Edit the Google Sheet. Then either:

- **Wait** — the workflow runs automatically every day at 6 AM IST
- **Trigger manually** — Actions → Sync menu from Google Sheets → Run workflow

The site updates within ~1 minute of the workflow finishing.

## CSV columns

| Column | Notes |
|---|---|
| `Section` | Section display name, e.g. `Biryani` |
| `Menu item` | Item name |
| `Price` | e.g. `129` or `129/229` for half/full |
| `Sort order` | Optional number. Empty = natural CSV row order, placed after sorted items |
| `Available` | `T`/`t` = show on menu · `F`/`f` or empty = hide |

---

## Changing the sync schedule

Edit the `cron` line in `.github/workflows/sync-menu.yml`.
Times are in UTC. IST = UTC+5:30.

```yaml
# Every day at 6 AM IST (00:30 UTC)
- cron: '30 0 * * *'

# Every 6 hours
- cron: '0 */6 * * *'

# Every Monday at 9 AM IST (03:30 UTC)
- cron: '30 3 * * 1'
```
